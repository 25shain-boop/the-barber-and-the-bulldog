# Anleitung – Barbershop One‑Pager

Dieses Repository enthält eine einfache einseitige Website (One‑Pager) für einen Barbershop. Die Dateien sind vollständig statisch und können kostenlos auf jeder Plattform gehostet werden, die statische Inhalte unterstützt (z. B. GitHub Pages, Netlify oder Vercel). Die Inhalte sind in deutscher Sprache gehalten und lassen sich leicht anpassen.

## Inhalt

- `index.html` – die Startseite mit allen Sektionen (Über mich, Dienstleistungen, Preise, Kontakt) inklusive einer eingebetteten Logo‑Grafik im Header.
- `style.css` – das dazugehörige Stylesheet mit Farbvariablen und Layout. Die Farbpalette orientiert sich am Logo (dunkle Basistöne und goldene Akzente).
- `images/logo.png` – das Logo „The Barber & The Bulldog“, das automatisch im Header eingebunden wird.

## Nutzung

1. **Entpacken:** Öffnen Sie die Dateien in einem Code‑Editor wie VS Code.
2. **Anpassen:** Ersetzen Sie die Beispieltexte (z. B. Adresse, Preise und Öffnungszeiten) durch Ihre realen Angaben. Passen Sie bei Bedarf Farben in der `style.css` an.
3. **Testen:** Öffnen Sie `index.html` in Ihrem Browser und prüfen Sie das Ergebnis.

## Kostenloses Hosting

Um den One‑Pager ohne monatliche Gebühren zu veröffentlichen, können Sie einen Dienst wie GitHub Pages nutzen. Legen Sie einfach ein GitHub‑Repository an, fügen Sie die Website‑Dateien hinzu, aktivieren Sie GitHub Pages in den Repository‑Einstellungen und Ihre Seite ist online.

## Lizenzhinweis

Die bereitgestellten Dateien dienen als Vorlage und dürfen beliebig angepasst und weiterverwendet werden. Bitte ersetzen Sie die Platzhalter durch Ihre eigenen Inhalte.
